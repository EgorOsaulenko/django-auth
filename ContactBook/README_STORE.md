# Інструкція по використанню магазину

## Встановлення та запуск

1. Створіть міграції:
```bash
python manage.py makemigrations
python manage.py migrate
```

2. Створіть суперкористувача (адміністратора):
```bash
python manage.py createsuperuser
```

3. Запустіть сервер:
```bash
python manage.py runserver
```

## Функціонал

### Для всіх користувачів:
- Головна сторінка з головними та топ товарами
- Реєстрація та вхід
- Перегляд товарів
- Додавання товарів до кошика (після входу)
- Перегляд кошика

### Для адміністраторів:
- Додавання нових товарів через форму
- Управління товарами (видалення)
- Управління головними та топ товарами через адмін панель (/admin/)
- Перегляд списків головних та топ товарів

## Використання адмін панелі

1. Перейдіть на `/admin/`
2. Увійдіть як суперкористувач
3. Додайте товари в моделі `Product`
4. Додайте товари до `FeaturedProduct` (головні товари) та `TopProduct` (товари у топі)
5. Встановіть порядок відображення через поле `order`

## Особливості реалізації

### Кешування
- Головна сторінка кешується на 15 хвилин
- Кеш автоматично очищається при додаванні/видаленні товарів

### Cookie
- Відстеження кількості відвідувань користувача

### Сесії
- Збереження товарів у кошику в сесії користувача
- Автоматичне оновлення при зміні кошика

### Сигнали
- Автоматичне видалення товарів з кошиків при видаленні товару
- Автоматичне очищення кешу при зміні товарів

### Захист адмін сторінок
- Сторінки додавання/управління товарами доступні тільки адміністраторам
- Автоматичне перенаправлення на сторінку входу для неавторизованих користувачів
